K · KOL
内容制造者
与社会证明
Awareness Layer
以 30 位腰部 KOL 为主力，90 位 KOC 铺量，2 位大博主背书。内容以「踩点」为核心——场景化穿搭 + 节拍转场，传递品牌调性而非产品功能。大 V 做背书，腰部做渗透，KOC 做真实感。
F · Feed
精准触达
与记忆强化
Recall Layer
通过聚光平台的信息流广告，将已互动用户（看过笔记、搜索过相关词）纳入再触达池。Feed 的核心作用不是拉新，而是让「见过一次」变成「多次出现」，在用户心中建立品牌存在感权重。
S · Search
决策验证
与信任锚点
Validation Layer
布局核心搜索词（品牌名 + 品类词 + 场景词），确保用户主动搜索时能看到足量正向内容。通过小红星进行搜索广告投放，抢占竞品关键词，植入真实 UGC，强化「别人也在用」的信任逻辑。
L · Live
临门转化
与情感关闭
Conversion Layer
直播是在用户已有认知基础上完成「最后一步说服」。内容策略以「真实试穿 + 场景模拟 + 限时权益」为核心。通过聚光 DLive 对高意向人群定向推流，确保进场用户已完成初步心智建立。
策略决策依据
预算分配：KOL 40%（RMB 400,000）/ Feed 25%（RMB 250,000）/ Search 20%（RMB 200,000）/ Live 15%（RMB 150,000）。前期（1-4周）集中 KOL 与 Search，确保内容密度先达标；中期（5-8周）启动 Feed 再触达；后期（9-12周）Live 转化收割。任何单独节点的孤立投放，ROI 均低于整体协同的 60%。

04
Channel Strategy

三套工具，
各司其职

聚光、小红星、直播投流——每个工具都有明确的战略定位，不做功能重叠的冗余投放。

聚光平台
(Juguang)
付费分发
精准触达 + 内容加热
  • 信息流广告（CPM / CPE）：对兴趣标签人群定向投放，覆盖时尚、运动、Z世代兴趣圈层
  • 笔记加热（Feeds Boost）：遴选自然互动率 Top 20% 的 KOL 笔记进行付费扩量
  • 再营销受众：将「收藏 / 评论 / 点击主页」的高意向用户打入再触达池
  • DLive 直播推流：直播进行时向高匹配人群精准推送，提升直播间 UV 质量
决策逻辑
聚光的核心价值不是拉新，而是在已有内容基础上做「精准放大」。在内容密度不足时盲目投聚光，等于烧钱提速烧光。
小红星
(搜索广告)
搜索截流
决策节点拦截 + 竞品防御
  • 品牌词投放：确保用户搜索品牌名时，搜索结果页第一屏被品牌内容占据
  • 品类词布局：覆盖「美国鞋品牌」「小众球鞋」等高意向品类词
  • 竞品词拦截：针对同赛道竞品关键词投放，将主动搜索流量截流至本品牌内容
  • 场景词长尾：「春季穿搭推荐」「通勤鞋」等场景词，捕捉高转化意图用户
决策逻辑
小红书搜索行为是用户主动发出的「我想了解」信号——这类流量的转化意图远高于被动触达。搜索广告是将「有意向的人」精准引导到品牌内容的最低成本路径。
直播投流
(DLive)
转化收割
高意向流量汇聚 + 即时决策激活
  • 前置内容联动：直播前 3 天，KOL 发布「直播预告」笔记并进行聚光加热
  • 定向进场人群：通过 DLive 对「收藏过相关笔记」「搜索过品牌词」的用户推送入口
  • 直播内容结构：真实试穿（信任）→ 场景演示（想象）→ 限时专属权益（行动）
  • 数据实时优化：监控 GPM 和 UV 价值，实时调整投流出价策略
决策逻辑
直播的作用是「临门一脚」而非「开门引流」。进入直播间的用户应该已经完成初步心智建立——否则转化率会因信任不足而低于行业均值。

06
策略布局 · 策略决策

我的参与方式：
决策，而非执行

顾问的价值不在于帮客户「做」，而在于帮客户「想清楚」——确保每一分预算都花在经过验证的路径上。

01
策略框架设计
在项目启动前，完成目标受众路径分析、KOL 层级筛选逻辑、内容主题框架、投放工具组合方案及预算分配模型（总预算 RMB 1,000,000 的最优分配路径）。确保执行团队有清晰的战略地图，而非依赖经验拼凑动作。
02
阶段决策节点把控
每阶段结束前，基于数据进行「继续 / 调整 / 停止」的三选一判断，并给出具体调整方向。重点关注：内容密度是否达标、搜索词收录率是否满足进入下一阶段的条件，以及人群质量是否符合预期。
03
KOL 筛选与内容 Brief
提供 KOL 评估标准（非粉丝量，而是圈层匹配度、商业合作频次、笔记互动质量）以及内容 Brief 框架，确保 122 位达人矩阵的内容在保留真实感的同时，精准传递品牌调性与穿着场景信息。
04
数据解读与优化建议
每周提供数据解读报告（非数字罗列，而是基于数据的战略建议），识别哪些信号值得追投、哪些路径应该收缩，以及下一阶段的预算再分配逻辑。将数据转化为可执行的决策，而非停留在分析层。
